# Academic Writing Style Guide

Write in a fluid, high-quality academic style:

- Never use bullet points or numbered lists in the text. Instead, write in well-structured paragraphs.
- YOU MUST avoid overused academic words such as 'nuanced', 'inherent', 'fundamental', 'critical', 'systematically', 'subtle'.
- Present ideas in a natural flow, connecting each paragraph smoothly to the next.
- Express complex ideas through clear, well-crafted sentences rather than relying on bullet points.

## Key Requirements

1. **LaTeX Syntax**: Use proper LaTeX syntax (no markdown)
2. **Citations**: Use `\cite{}` for citations
3. **Tables**: Use proper LaTeX tabular environment
4. **Figures**:
   - Use complete figure environment with proper captions
   - Use simple filenames without paths (e.g., 'figure1.png' instead of 'path/to/figure1.png')
   - Always include `\label{}` for cross-referencing
5. **No Placeholders**: No placeholders or TODOs
6. **Text Emphasis**:
   - Use `\textbf{}` for bold text (never use **, * or other markdown syntax)
   - Use `\textit{}` for italics
   - Use `\emph{}` for emphasis within italicized text
7. **Professional Style**: Professional academic writing style
8. **Subsections**:
   - Use proper LaTeX `\subsection{}` syntax
   - Each subsection must contain at least 2 well-developed paragraphs
   - For Related Work section, limit to maximum 3 subsections
   - Ensure smooth transitions between subsections
9. **Citations**: All citations MUST reference actual papers from the provided bibtex file - do not make up or reference non-existent papers
10. **LaTeX Formatting**:
    - Use proper LaTeX quotation marks ('') instead of straight quotes
    - Avoid non-LaTeX punctuation marks
    - Use proper LaTeX dashes (-- for en-dash, --- for em-dash)
    - Use `\dots` for ellipsis instead of ...
