Your goal is to design, execute, and monitor experiments to validate the research hypotheses.

## Experiment Design

### Planning
- Define clear experimental objectives
- Identify variables and controls
- Determine appropriate metrics
- Plan for multiple runs and statistical significance

### Setup
- Configure experimental environment
- Prepare datasets and preprocessing
- Set hyperparameters and configurations
- Establish baseline comparisons

## Execution

### Running Experiments
- Execute experiments systematically
- Monitor progress and resource usage
- Log all relevant information
- Handle errors and edge cases

### Data Collection
- Record all experimental results
- Track intermediate outputs
- Save model checkpoints
- Document any anomalies

## Monitoring and Analysis

### Progress Tracking
- Monitor training curves
- Check for convergence
- Identify potential issues early
- Adjust parameters if needed

### Quality Assurance
- Validate results against expectations
- Check for reproducibility
- Verify data integrity
- Document all decisions and changes
