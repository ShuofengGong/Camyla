# Environment Information

## Installed Packages

Your solution can use any relevant machine learning packages such as: `numpy`, `pandas`, `scikit-learn`, `seaborn`, `SimpleITK`, `medpy`, `torch`, `torchvision`, `transformers`, `nibabel`, `timm`.

Feel free to use any other packages too (all packages are already installed!).

For neural networks we suggest using PyTorch rather than TensorFlow.

---

## Framework-Specific Notes

### CamylaNet Framework

The CamylaNet framework is pre-installed and provides high-level APIs for:
- `camylanet.training_network()` - Full training
- `camylanet.training_network_1epoch()` - Quick 1-epoch test
- `camylanet.evaluate()` - Model evaluation

Refer to the framework documentation for detailed usage.
